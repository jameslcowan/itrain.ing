JSON → share link

Paste program JSON from an LLM, spreadsheet export, or script. This page compresses it the same way the app does and gives you a /program/… URL you can open or send to someone.

How to use

  1. Paste JSON in the box below (or click Insert example to try a sample program).
  2. Click Generate link. Fix any errors shown in red if the JSON is invalid or missing required fields.
  3. Copy the “Clean link” (/program/…) and share it. Recipients open it in the builder with your exact program.

Tip: Ctrl+Enter (or ⌘+Enter) runs Generate link. Everything runs locally in your browser — nothing is uploaded. Legacy /#/p/… links are listed for reference only.

  "v": 1,
  "u": "lb" | "kg",
  "c": [{ "n": "Meso 1" }],
