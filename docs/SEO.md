# SEO and positioning

## Positioning

**powerlift.ing** is the **best free powerlifting program builder**.

Build an SBD program fast, then share it as one link.

- Free, no signup
- Mobile-first
- Coach / training-partner friendly

Use natural powerlifting terms (don’t spam): powerlifting program, program builder, mesocycle, peaking, meet prep, RPE, %1RM, squat bench deadlift.

**Creator:** James L. Cowan — [@jameslcowan](https://x.com/jameslcowan)

## Indexing

| Index | Path |
|-------|------|
| Yes | `/`, `/blog/`, `/blog/*`, `/faq/`, `/programs/`, `/terms/`, `/privacy/` |
| No | `/app`, `/app/*`, `/program/*`, `/p/*` (user-generated, effectively infinite) |

Enforced via `netlify.toml` headers on shared-program routes. Marketing pages use normal meta + canonical from `renderShell`.

## URL and state contract

- **Current:** `/app/<STATE>`
- **Legacy (still work):** `/p/<STATE>`, `/#/p/<STATE>`, `/program/<STATE>`

The builder updates the URL with `history.replaceState`. Old shared links must keep working.
